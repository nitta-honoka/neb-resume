# neb-resume

### 基于星云链和 vue 的自动简历生成器

>

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8084
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
